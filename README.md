# Drugs 4U - Pharmacy Management System

## Group 1

### Overview

Drugs 4U is a web-based Pharmacy Management System developed as part of the COMP70047 Enterprise Systems module at the University of Staffordshire.

The system was designed to help pharmacies digitize their prescription, inventory, and customer management processes while improving operational efficiency and patient safety.

---

## Features

### Customer Management

* Register new customers
* Maintain customer records
* Store customer information including:

  * Name
  * Address
  * Date of Birth
  * Contact Details

### Prescription Management

* Upload prescriptions
* View prescription history
* Manage prescription records

### Sales & Billing

* Create sales invoices
* Generate customer bills
* Print invoices
* Track medicine purchases

### Inventory Management

* Add and manage medicines
* Monitor stock levels
* Update inventory quantities
* Maintain medicine information

### Vendor Management

* Manage supplier information
* Create purchase orders
* Print purchase orders
* Track medicine procurement

### Dashboard & Reporting

* Dashboard overview
* Customer reports
* Prescription reports
* Inventory monitoring

### Risk & Alert Management

* Low stock alerts
* Inventory monitoring
* Prescription verification support

## Installation Guide

1. Install XAMPP.
2. Start Apache and MySQL services.
3. Copy the project folder into:

C:\xampp\htdocs\pharma.lk

4. Create a MySQL database named:

pharmacy

5. Import:

Database/pharmacy.sql

6. Open your browser and navigate to:

http://localhost/pharmacy
